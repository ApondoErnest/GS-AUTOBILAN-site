# V1 / V2 boundary

**Step S006** · Prevents scope creep.

## V1 (one sentence)

Professional bilingual website + admin + **booking intake** + **appointment/document tracking**.

## Not in V1 booking/tracking statuses

Vehicle registered · inspection in progress/completed · result under validation · accepted/suspended/refused · certificate ready · counter-visit pending · lane/machine data  

*(Accepté / Suspendu / Refusé may appear only as education on Visite Technique.)*

## V2 only with real integration / explicit re-scope

SMS/WhatsApp APIs · Mobile Money · customer/fleet portals · certificate/QR · operational counter-visit · advanced reporting · **real lane tracking** (requires ops software/API)

## Change process

1. Write request + reason  
2. Update this file + [02-scope.md](02-scope.md)  
3. Adjust [../../STEPS.md](../../STEPS.md) only if approved
