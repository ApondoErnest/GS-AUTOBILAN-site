## 2. Colour palette

| Purpose | Colour | Hex |
|---------|--------|-----|
| Primary blue | Deep technical blue | `#0B3A75` |
| Dark background | Navy | `#061A33` |
| Accent strip | Ox-blood red | `#7A1621` |
| Main background | White | `#FFFFFF` |
| Section background | Soft grey | `#F4F7FB` |
| Main text | Charcoal | `#1F2937` |
| Muted text | Grey | `#6B7280` |
| Success | Green | `#15803D` |
| Warning | Amber | `#D97706` |
| Error | Red | `#B91C1C` |

**Ratio:** blue ≈60% · white ≈30% · ox-blood ≤10%.

### Application

- Top strip: ox-blood  
- Header: deep blue (or white on light variants)  
- Primary buttons: blue  
- Secondary: white + blue border  
- Section accents: ox-blood  
- Footer: navy  
- Cards: white · Backgrounds: soft grey  
- Status badges: green / amber / red by meaning  

---

## 3. Typography

Bold headings · clean body · generous line height · clear hierarchy · strong CTAs · large mobile form labels (≈44px touch targets).

Prefer a clean modern web font until the company specifies brand fonts. Avoid generic “AI default” purple/cream newspaper looks — stay on the blue / white / ox-blood system.
